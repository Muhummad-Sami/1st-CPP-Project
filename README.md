# 🛒 Grocery Store Bill Management System

## 📖 Description
A console-based C++ application that simulates a grocery store billing system. It allows users to select products, enter quantities, and automatically generates a bill with discounts.

## 🚀 Features
- Menu-driven product selection  
- Quantity-based billing  
- Automatic discount calculation  
- Customer name input  
- Bill summary display  
- File handling to save bill (`sample.txt`)  

## 🛠️ Technologies Used
- C++  
- File Handling (ofstream)  

## 🧠 Concepts Applied
- Arrays & Pointers  
- Functions  
- Switch Case & Loops  
- Conditional Statements  
- File Handling
